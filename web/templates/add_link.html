How it works

When you add a new link, the system will:

  • Fetch the page and extract metadata (title, author, excerpt)
  • Check if the URL is accessible
  • Extract the domain and create a quality score
  • Save the link to your collection

You can later convert the link to markdown or refresh its metadata.
